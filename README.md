Assembler_Simulator
===================

INSTRUCTIONS

Run the "TextFrame" and place the location of a .txt file with assembly code and click ok . The program will compile and show the result ;



OPERATIONS SUPPORTED
  - add
  - sub
  - and	
  - or
  - not
  - nor
  - slt
  - addi
  - subi
  - andi
  - ori
  - slti
  - lw
  - sw
